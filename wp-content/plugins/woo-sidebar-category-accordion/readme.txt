== Description ==

Turn your standard woocommerce product categories widget into a smooth collapsible accordion.
This plugin should work out the box with most themes.

* Supports unlimited subcategories
* Current category will be expanded on page load
* Lightweight and requires little configuration
* You can customise the look in your themes CSS editor

== Installation ==

Once installed the plugin will work wherever you decide to use your woocommerce product categories widget.

* Upload and install the plugin
* Make sure you have the woocommerce "Product Categories" widget enabled in your sidebar
* Un-check all options except for "Show hierarchy"
* Save the widget's configurations
