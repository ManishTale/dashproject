Add Cancel order request button at my account page.2. Customer can send order cancellation request to the store manager or admin.3. Store manager or admin can approve or deny the order cancellation request.4. Auto Restore product inventory/stock when cancel request accepted.> <strong>Get following features/options with WC Cancel Order Pro:</strong><br>> * Confirmation alert before send cancellation request.<br>> * Know cancellation detail/reason using popup box.<br>> * Display custom confirmation message.<br>> * Configurable from back-end setting section.<br>> * Send e-mail notification to admin/customer.<br>> * Multilingual ready / Translation ready.<br>> * Full live/Pro Support.<br><br>> <strong>[Get WC Cancel Order Pro:](http://wooexperts.com/product/wc-cancel-order-pro/)</strong><br>== Installation ==1. Simply install and activate the plugin.2. After installation the order cancel request button will appear at woocommerce my account page.3.
